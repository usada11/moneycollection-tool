# Money Collection Tool
集金管理・支払い確認を簡単に行うための Web アプリ。

# Features
- 支払い状況の確認
- メンバー管理
- 集金状況の可視化
- 未払いチェック
# Tech Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Vercel
# Requirements
- Node.js 18+
- npm
# Installation
bash


> git clone https://github.com/usada11/moneycollection-tool.git
> cd moneycollection-tool
> npm ci
# Development

> npm run dev
ブラウザで http://localhost:3000 を開いてください．
編集は app/page.tsx などを変更すると自動で反映されます．

# Build / Production

> npm run build
> npm run start
